# Qadsiah Predictions — World Cup 2026

QR-code score prediction site for Al-Qadsiah FC's match-viewing venue: scan, enter name + phone, predict the Saudi team's score. Exact-score winners go on a live lucky wheel that picks the prize winner.

See `CLAUDE.md` for full architecture, data model, security rules and setup instructions.

```bash
npm install
cp .env.example .env   # fill in Firebase web config
npm run dev
```
